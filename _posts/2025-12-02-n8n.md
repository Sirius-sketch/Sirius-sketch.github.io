## 构建环境
1.创建一个n8n文件夹，包含Dockerfile和docker-compose.yml

```dockerfile
FROM n8nio/n8n:latest

USER root

# 1. 使用 apk 安装 python3 和 pip
# build-base 和 python3-dev 是为了防止 PyMuPDF 编译时缺依赖
RUN apk add --update --no-cache python3 py3-pip build-base python3-dev

# 2. 安装 Python 库
# --break-system-packages 是必须的，因为新版 Python 保护机制限制直接 pip install
RUN pip3 install PyMuPDF pypdf --break-system-packages

USER node
```


```yaml
version: '3'

services:
  n8n:
    build: .
    ports:
      - "5678:5678"
    environment:
      - N8N_HOST=localhost
      - N8N_PORT=5678
      - N8N_PROTOCOL=http
      - GENERIC_TIMEZONE=Asia/Shanghai
    volumes:
      - ~/.n8n:/home/node/.n8n
```


2.构建镜像：docker compose build  
启动：n8ndocker compose up -d（之后可直接在docker desktop里面启动）  
在浏览器打开http://localhost:5678  

3.在Credentials中添加API key，我用的moonshot（base URL填https://api.moonshot.cn/v1/
）和github API（获取：GitHub Settings -> Developer settings -> Personal access tokens -> Generate new token (Classic)）
